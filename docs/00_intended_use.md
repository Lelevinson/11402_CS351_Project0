# 00 - Intended Use

## Purpose

This project demonstrates and compares two C++ solutions to the Two Sum problem.
It is intended for algorithm practice, introductory software engineering workflow, and test-driven development exercises.

## Problem Statement

Given an integer array and a target value, find two distinct indices such that the values at those indices add up to the target.

## Target Users

- Students learning data structures and algorithms.
- Instructors evaluating correctness, complexity, and coding quality.
- Developers who want a simple baseline for comparing brute-force and hash-table approaches.

## Intended Environment

- Local development on Windows, macOS, or Linux.
- C++20 toolchain (e.g., GCC, Clang, MSVC).
- Source code under `src/`, tests under `tests/`, CI in `.github/workflows/`.

## In Scope

- Implement Two Sum using two distinct methods.
- Provide clear algorithmic trade-off documentation.
- Provide repeatable test strategy and acceptance criteria.

## Out of Scope

- Full benchmarking suite with statistical analysis.
- GUI or web API.
- Persistent data storage.

## Success Criteria

- Both methods compile and run.
- Functional requirements in SRS are satisfied.
- Acceptance tests pass.
- Documentation supports maintainability and grading.
